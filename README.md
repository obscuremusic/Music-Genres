##### Layout
###### The entries in the list are structured with the following layout.
```html
    ┌ Type 1
    ├ Genre 1
    │      ├ subgenre 1
    │      └ subgenre 2 ┐
    │                               ├ microgenre 1
    │                               └ microgenre 2
    └ Genre 1
             ├ subgenre 1
             └ subgenre 2 ┐
                                    ├ microgenre 1
                                    └ microgenre 2
     
    ┌ Type 2
    ├ Genre 1
    │      ├ subgenre 1
    │      └ subgenre 2 ┐
    │                             ├ microgenre 1
    │                             └ microgenre 2
    └ Genre 1
             ├ subgenre 1
             └ subgenre 2 ┐
                                    ├ microgenre 1
                                    └ microgenre 2
```
